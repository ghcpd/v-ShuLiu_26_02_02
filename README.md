# prettytable-pr78-demo

Small Python 3-only demo project focused on maintenance tasks: cleaning up packaging metadata, CI target versions, and tooling configuration while keeping business behavior unchanged.

## Setup

Create and activate a virtual environment (PowerShell):

1. `python -m venv .venv`
2. `.venv\Scripts\Activate.ps1`

Install dependencies:

- `pip install -r requirements.txt`

## Self-checks

Run the test suite:

- `pytest`
